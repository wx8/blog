不想写标题了

『 熊的猫写于 』

今天我的博客在祖国大陆地区又可以重见天日了，心情格外美丽。其实在这段时间里，我的博客一直都可以访问，只不过你需要把自己的IP弄到国外去。如今媳妇熬成婆，blog.ps 终于又以胜利者的姿态回归了！借着这个机会，我可以先简单说一下博客与我的近况。免得大家误以为我又写着写着不写了，然后跑路了。

博客备案了

作为一个有理想有抱负，实现未来共产主义的有效分母之一，我的博客怎能不积极响应国家的号召呢？所以就在大年初六便匆匆地向工信部提交了备案申请。为啥是匆匆的呢？因为不抓紧时间的话，备案要是被拖入3月份，那等待就更加漫长了。毕竟3月份有一次全国的大表演在那等着呢，所以会很费时间的。

没想到的是，可能审核备案的人也想着急看表演，所以备案环节很顺畅，1周的样子就下来了，这效率出乎我的意料啊。

博客换程序了

嗯，话说从 wordpress 换到 typecho 才1个月的时间，我又决定开始新一轮的折腾——再次更换程序——反正备案期间很少有人能来访问我的博客，所以可以尽情的折腾了。新的博客采用纯静态博客搭建，也不用考虑后台、数据库、安全等等，每一页就是一个真实的html页面了。同时，电脑上还有数据备份，应该会很靠谱了。

说道静态博客，我就得多唠叨两句。作为一个完完全全的小白，静态博客生成器在本地的搭建工作对于我来说很头大。不怕你笑话，春节期间，在家养病的我还几天几夜的投入在搭建 Node.js 服务器环境的大坑里。虽然我找着网上的各种教程而走的小心翼翼，但还是精准的跳进了每一个前人曾经掉进过的坑……

后来，偶然的机会在小众软件网站上，看见有人推荐纸小墨，下载回来后发现这正是我想要的——几行简单的代码就可以傻瓜化地生成静态博客页面了，感觉好幸福。

博客搬家了

程序啥的都换了，那么我自然没必要再花大价钱租赁虚拟主机、服务器之类的了。网上普遍都推荐静态博客托管在 github 的 page 服务上，免费不限流量，但是却有两个缺点。一个是速度不太快，毕竟是国外的服务商嘛；另一个则是想要上https只有一种解决方法，就是用 cloudflare 的解析服务。而在咱们祖国， cloudflare 时不时就被艹几下，让人忍无可忍。

这个时候想起了七牛和又拍云，速度倒是飞一般的感觉，但是这两家的存储空间自带 CDN，网站不能实时更新，也不行。

最后选择了 Coding 的 page 服务。服务标准上，Coding 和 github 一样，还能一键生成 Let’s Encrypt 证书，完美。
