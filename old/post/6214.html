说一说DNS

『 熊的猫写于 』

闲来无事又不知道要写点什么，于是就把目光转投到域名的设置上了。 首先，我把固有的DNS给品牌化了，我现在的博客DNS服务器已经变成了ns8.blog.show和ns9.blog.show，看起来是不是很高大上呢？

以往想要实现这样的方法，要么花大价钱和技术自己架设DNS，要么就要单独购买这样的服务（CloudFlare的价格是200美元/月）。但是好在现在有了Vanity Name Servers技术。用Google查了一下，国外还是有几家可以免费支持虚拟的DNS服务的。

Zonomi：免费账号最多可添加1个域名、10条解析记录，每个月1百万查询量。他家付费解析倒也不贵，10美元1年。 DigitalOcean：恐怕大家都知道他家的VPS和优惠力度，其实他家的DNS解析系统也非常不错。DNS系统走的是CloudFlare，免费支持Vanity Name Servers，一下子每个月就省了200美元的庞大支出。他家免费开通账号时需要信用卡或者PayPal 30美元的验证费（费用可以购买VPS）。 Vultr：有了DO，就不能落下其竞争对手。Vultr的DNS管理系统也支持Vanity Name Servers，免费！ RackSpace：也是免费支持Vanity Name Servers，不过和DO一样，开户要绑信用卡，还不支持Paypal。所以没做过多研究。 dnsmanager.io：这是一个来源不详的解析系统，可以支持Vanity Name Servers。我查了一下，似乎2014年开始就有这个DNS解析系统了。大陆速度比较慢。

其实Vanity Name Servers这东西就是面上的活儿，除了好看也没啥。要想得到更强大的DNS解析功能，还需要Google一下custom dns nameservers。

因为换了DNS的服务器（目前我用的是Vultr的DNS服务），顺便也把DNSSEC和CAA记录也添加进域名中了。虽然还不知道这俩玩艺到底有什么作用，但与时俱进跟跟时髦的技术总是好的。
